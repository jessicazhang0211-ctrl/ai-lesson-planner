## 一、作品（实物）说明

本作品为“AI Lesson Planner”智能教学设计与学习管理系统，是一个面向教师备课、班级管理、学生练习和学习反馈的 Web 应用系统。系统围绕教师教学准备、资源发布、学生练习提交、教师评阅与学习分析等教学流程展开，旨在提升教师生成教学内容和管理学习过程的效率，同时为学生提供较为清晰的学习任务入口与练习反馈。

系统采用前后端分离架构。后端基于 Python Flask 框架开发，使用 Flask-SQLAlchemy 进行数据持久化管理，数据库采用 MySQL；前端采用 Vite 与 React 构建应用入口，并兼容教师端、学生端已有的 HTML、CSS、JavaScript 页面。系统通过 RESTful API 完成前后端数据交互，并使用 JWT 实现用户认证与权限控制。

作品主要包含以下功能模块：

1. 用户认证模块：支持教师注册、教师登录、学生登录、用户资料维护、密码修改等功能。
2. 班级管理模块：支持教师创建班级、查询班级、导入学生、管理学生状态、重置学生密码和查看班级统计信息。
3. AI 教案生成模块：支持教师输入教学主题、年级、知识点等信息，调用大语言模型生成结构化教案，并提供历史记录、版本查看、修改与回滚能力。
4. AI 习题生成模块：支持根据知识点和教学目标生成练习题，提供习题历史、自适应练习方案和变式题生成能力。
5. 资源发布与评阅模块：支持教师发布教学资源或练习任务，学生完成提交后，教师可进行查看、评分与反馈。
6. 学生学习模块：支持学生查看作业、完成练习、查看成绩、回顾练习记录，并获得学习情况分析。
7. 知识库与统计模块：支持知识点数据导入、资源统计、学生知识掌握情况分析和学习建议生成。
8. 测试与模型评估模块：项目内置自动化测试脚本、提示词测试数据和 AI 模型实验脚本，可用于验证系统基础功能和比较不同模型的生成效果。

系统的主要特点如下：

1. 将 AI 生成能力嵌入教学设计流程，辅助教师快速生成教案和习题。
2. 兼顾教师端与学生端，形成从备课、发布、练习、提交到评阅反馈的教学闭环。
3. 使用数据库保存用户、班级、教案、习题、资源发布、提交记录和学习分析数据，便于后续查询和统计。
4. 提供自动化测试与模型实验材料，可支撑毕业设计论文中的系统测试和实验分析章节。
5. 项目结构清晰，前后端模块划分明确，具备继续扩展和部署的基础。
