# AItest 所需 API Key 官方网址

更新时间：2026-03-15
说明：以下为常用官方入口，请优先使用官方控制台创建密钥。

## 核心实验组（推荐）

1. OpenAI
- 平台主页：https://platform.openai.com/
- API Keys 页面：https://platform.openai.com/api-keys

2. Anthropic Claude
- 控制台主页：https://console.anthropic.com/
- Keys 页面：https://console.anthropic.com/settings/keys

3. Google Gemini
- Google AI Studio：https://aistudio.google.com/
- Gemini API 文档入口：https://ai.google.dev/

4. DeepSeek
- 平台主页：https://platform.deepseek.com/
- API 管理页：https://platform.deepseek.com/api_keys

5. Qwen（阿里云 DashScope）
- DashScope 控制台：https://dashscope.console.aliyun.com/
- API Key 管理（控制台内）：https://dashscope.console.aliyun.com/apiKey

## 可选扩展组

1. xAI Grok
- 控制台：https://console.x.ai/

2. Mistral AI
- 控制台：https://console.mistral.ai/

3. Cohere
- 控制台：https://dashboard.cohere.com/

4. Zhipu GLM
- 开放平台：https://open.bigmodel.cn/

5. Moonshot Kimi
- 开放平台：https://platform.moonshot.cn/

## 使用提醒

1. API Key 不要写入前端代码和公开仓库。
2. 建议写入后端环境变量，例如 backend/.env。
3. 建议为不同实验批次使用不同 Key 并记录配额与成本。
